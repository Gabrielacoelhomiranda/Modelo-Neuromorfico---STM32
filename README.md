Este trabalho apresenta o desenvolvimento de um modelo embarcado utilizando o microcontrolador STM32 com o objetivo de biomimetizar o comportamento de neurônios de adaptação lenta e de adaptação rápida. O sistema implementa modelos neuronais capazes de reproduzir diferentes padrões de disparo em resposta a estímulos externos, permitindo a análise comparativa entre os dois tipos de adaptação. A abordagem embarcada possibilita a execução em tempo real, com potencial aplicação em sistemas neuromórficos, interfaces neurais e dispositivos biomédicos inteligentes.
